NEXT STEPS FOR CONSUMING SERVICES
================================================================================

   <dependency>
       <groupId>io.theawesomemogul</groupId>
       <artifactId>mogul-audio-core</artifactId>
       <version>1.0.0-SNAPSHOT</version>
   </dependency>

2. Add to mogul-LLM-engine pom.xml:
   <dependency>
       <groupId>io.theawesomemogul</groupId>
       <artifactId>mogul-audio-core</artifactId>
       <version>1.0.0-SNAPSHOT</version>
   </dependency>

3. Extend BaseEntity or SoftDeletableEntity in your domain models
4. Use I18nService for internationalized messages
5. Use MogulLogger for structured logging
6. Use exception classes for consistent error handling
7. Use SecurityUtils to access authentication context
8. Use utility classes for common operations
